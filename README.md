# Last.fm Backup

### Features:
* Simple
* One format for export data:
    1. `dump` - JSON, direct from API

### How to use:
* Stop scrobbling!
* Modify `config.json`.
* [Get API Key](http://www.last.fm/api/account/create).
* Run `lastfm_backup.py`.
* WAIT =)
